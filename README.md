# Token payment - v1

##### 1. Install node_modules:

`npm install`

##### 2. Start Dev:

`npm run start`

##### 3. Start production:

`npm run start`

##### 4. open project

`http://localhost:3000`

## Project structure:

    ./token-payment
     ./build
     ./.vscode                        || Folder chứa các setting cho Visual Studio Code
     ./public
     ./src
        ./ethereum
        ./redux
        ./components                  || Chứa các folder component, trong mỗi folder là các file component
        ./styles
     .env
     .gitignore
